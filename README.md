# Task List API

Live API : [ click to open ](http://todolistsapi.herokuapp.com).

## Tools used :

Restify Tasks API RESTful API built with Node.js, Restify, Mongoose and JWT.

## API Endpoints 

## Tasks Routes 

GET /tasks 

GET /tasks/:id 

POST (Protected) /words 

PUT (Protected) /tasks/:id 

DELETE (Protected) /tasks/:id

## User Routes 

POST /register 

POST /auth"# tasksapi"
